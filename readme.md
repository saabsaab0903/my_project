### This is a tutorial of using github

1. install git http://git-scm.com/
2. create account
3. clone from here
    1. git clone {url}
4. make some changes
5. create your own repository
6. upload to yours
    1. git status
    2. git add *
    3. git commit -m 'message'
    4. git push
    5. checkout your github
